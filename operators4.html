More Boolean Fun

Logical "OR" returns the result, so you can use it for defaulting values 

console.log(null || "not null") // -> "not null"
Ternary operator "?:", basically acting like an "if X then Y otherwise Z" 

const x = undefined
console.log(x ? "is defined" : "is undefined") // -> "undefined"
const luck = 888
console.log(luck > 100 ? "much luck" : "could be better") // -> "much luck"
