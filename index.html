Operators

Some operators that are useful for numbers: 

console.log(1 +  1) // -> 2
console.log(1 - 1) // -> 0
console.log(7 * 2) // -> 15
console.log(7 / 2) // -> 3.5
console.log(7 % 2) // -> 1
All these have shorthands for calculation and assignment 

const kickDamage = 10
let health = 100
health -= kickdamage // same as health = health - kickdamage
console.log(health) // -> 90
