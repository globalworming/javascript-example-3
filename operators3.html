Operators

Boolean negation with "!" can also have some surprising results 

console.log(!true) // -> false
console.log(!false) // -> true
console.log(!0) // -> true
console.log(!1) // -> false
console.log(!-1) // -> false
console.log(!undefined) // -> true
console.log(!null) // -> true
console.log(![]) // -> false
console.log(!{}) // -> false
console.log(!" ") // -> false
console.log(!"") // -> true
